# AI Hub - Responsive Frontend Interface

## Project Overview
AI Hub is a modern, responsive landing page built using HTML5, CSS3, and JavaScript. It showcases a clean UI with interactive elements and works smoothly across desktop, tablet, and mobile devices.

## Features
- Responsive Design
- Modern Navigation Bar
- Hero Section
- About Section
- Services Section
- Statistics Section
- Testimonials
- Contact Form with Validation
- Smooth Scrolling
- Hover Effects & Animations
- Sticky Navbar

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## Project Structure
```
AI-Hub/
│── index.html
│── style.css
│── script.js
│── README.md
```

## How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it using Live Server in VS Code.
